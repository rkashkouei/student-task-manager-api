# Student Task Manager API

A simple backend API built with Node.js and Express.

## Features

- Create a task
- Get all tasks
- Update a task
- Delete a task

## Technologies

- Node.js
- Express.js

## How to run

```bash
npm install
node index.js
```

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | / | Test API |
| GET | /tasks | Get all tasks |
| POST | /tasks | Create task |
| PUT | /tasks/:id | Update task |
| DELETE | /tasks/:id | Delete task |
